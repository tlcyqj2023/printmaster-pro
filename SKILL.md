## 三、预检核查清单（交付前必须逐项打勾）

### 必查项

```
□ 出血位：封面每边≥3mm，制作尺寸正确
□ 分辨率：图片≥300dpi（线条图≥900dpi）
□ 色彩模式：全部CMYK，无RGB颜色
□ 字体：嵌入所有字体，或转曲线（Outlines）
□ 表格：跨页有表头重复，无断行
□ 页边距：上下≥15mm，左右≥20mm（无线胶装）
□ 页码：位置统一，格式正确
□ 黑色文字：单色黑（K=100, C=M=Y=0）
□ 出血内容：图片/底色超出成品边缘≥3mm
□ PDF标准：PDF/X-1a 或 PDF/X-4
□ 印张计算：总页数÷2=印张，优先XX.0或XX.5
□ 校对：有第二人核对文字内容
□ 叠印黑色：黑色文字勾选"套印"（Overprint）
□ 图片链接：所有链接图片有效（无红叉红字）
□ 安全区：文字/徽标距成品边缘≥5mm
□ 色差标注：专色注明Pantone色号
□ 角线：四角有角线（裁切线）标记
□ 文件命名：含客户名+品名+数量+日期
```

## 四、字体规范（印刷安全字体）

### 安全字体（印刷必备）
```
✅ 宋体（SimSun）——正文，最标准
✅ 仿宋（FangSong）——正文/公文
✅ 黑体（SimHei）——标题/副标题
✅ 微软雅黑（Microsoft YaHei）——屏幕+印刷双用
✅ 方正书宋、方正黑体（商业授权版）
✅ Times New Roman——英文正文
✅ Arial——英文标题
✅ Helvetica——国际通用
```

### 危险字体（严禁使用）
```
❌ 华文行楷、华文彩云、华文新魏
❌ 任何手写体、书法体
❌ 品牌定制字体（未购买授权）
❌ 屏幕字体（如某些游戏字体）
```

### 字体处理三选一
| 方案 | 操作 | 适用 |
|------|------|------|
| **嵌入字体** | Word导出PDF → 勾选"嵌入所有字体" | 有Adobe软件客户 |
| **转曲线** | AI/ID选中文字 → 对象 → 创建轮廓 | 少量文字 |
| **转PNG** | 文字栅格化为300dpi透明图片 | 整段文字区 |

## 五、色彩模式与CMYK

### RGB → CMYK 转换规则

```
步骤1：确认所有图片为RGB模式
步骤2：用 Photoshop/Illustrator 转换为 CMYK
步骤3：检查青(C)、洋红(M)、黄(Y)、黑(K)各自数值
步骤4：确认K=100的文字为单色黑（不叠加C/M/Y）
步骤5：检查总墨量（TC）不超过300%（易蹭脏）
```

### 常见问题颜色标准
| 用途 | C | M | Y | K | 色号参考 |
|------|---|---|---|---|---------|
| 纯黑（大面积） | 60 | 40 | 40 | 100 | Warm Black |
| 纯黑（文字） | 0 | 0 | 0 | 100 | Rich Black |
| 深蓝 | 100 | 80 | 0 | 30 | Pantone 287C |
| 红色 | 0 | 100 | 100 | 0 | Pantone 485C |
| 黄色 | 0 | 0 | 100 | 0 | Pantone YellowC |

## 六、表格跨页最佳实践

### 方案A：Word（最常用）

**操作：**
1. 选中表头行（必须包含首行）
2. 「表格工具」→「布局」→「重复标题行」✅
3. 设置行不允许跨页断行：
   - 右键表格 → 表格属性 → 行 → ✅"允许跨页断行"（取消勾选）

**验证：** 打印预览，跨页处应有表头

### 方案B：Excel → Word粘贴

1. Excel整理数据 → 全选 → Ctrl+C
2. Word光标定位 → 「开始」→「粘贴」→「只保留文本」
3. 插入表格 → 「将文本转换为表格」
4. 设置表头 → 重复标题行

### 方案C：InDesign（最高质量）

**操作：**
1. 文字工具拖出文本框 → 粘贴Excel内容
2. 「表」→「将文本转换为表」
3. 选中表头行 → 「表」→「转换行」→「到表头」
4. 溢出文本红色三角 → 拖到下页，自动产生跨页表头

**⚠️注意：** 改表头只能在首页改，其他串接页锁定

## 十一、案例：错误案例与正确做法

### 案例1：封面四周有条白边
**原因：** 制作尺寸=成品尺寸，未加出血
**正确：** 封面四周各外扩3mm → 制作216×303mm

### 案例2：表格第2页无表头
**原因：** 手工复制表头，删除行后错位
**正确：** Word「重复标题行」✅；ID「转换行→到表头」

### 案例3：客户打开PDF字体变了
**原因：** 字体未嵌入，客户机无此字体
**正确：** 导出PDF → 嵌入所有字体；或文字转曲线

### 案例4：颜色偏暗/偏红
**原因：** RGB模式直接印刷，CMYK转换后偏色
**正确：** 全部图片转CMYK后再插入文档

### 案例5：骑马钉钉不住
**原因：** 钉位距边缘太近（<3mm）
**正确：** 钉位距边缘≥5mm，上下各1钉

### 案例6：图片模糊
**原因：** Word截图分辨率不足96dpi
**正确：** 替换为≥300dpi原图

### 案例7：文字被裁切
**原因：** 文字紧贴成品边缘，裁切误差被切掉
**正确：** 文字距成品边缘≥5mm（安全区）
